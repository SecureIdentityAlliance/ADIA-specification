Title: VC Issuance Option1

Participant USER
Participant Issuer
Participant VC Storage
Participant DAP Agent
Participant Cloud Agent


USER -> Issuer: request issue VC 
Issuer -> DAP Agent: Invite the DAP Agent for DIDcomm conn
DAP Agent -> Issuer: DIDcomm connection estabilshed
Issuer -> DAP Agent: Queries for DID for user's DA (i.e. DAP_USER_ID)
DAP Agent -> Issuer: Responds with the DAP_USER_ID for DA

Issuer -> Issuer: Issue VC
Issuer -> VC Storage: Store VC
Issuer -> DAP Agent: request publish VC meta
DAP Agent -> DAP Agent: publish VC metadata
DAP Agent -> Issuer: publish metadata success
Issuer -> USER: VC issuance success
